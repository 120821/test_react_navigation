首先，让我们安装 expo-cli，一个用于处理 React Native 项目的命令行工具。
### `npm install --global expo-cli`
现在使用 expo-cli 创建一个新应用程序：
### `expo init MyApp --template blank`
### `cd MyApp`
### `npm start`
